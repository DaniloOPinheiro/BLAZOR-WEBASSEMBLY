# Incrível Blazor WebAssembly <img src = "https://adrientorris.github.io/wwwroot/images/blazor/logo-blazor.png" align = "right" width = "170">
> Uma coleção de recursos incríveis do Blazor.

Blazor é uma estrutura da web .NET usando C # / Razor e HTML que roda no navegador com WebAssembly.

Blazor WebAssembly: aplicativos de página única que são baixados para o navegador do cliente antes de serem executados. O tamanho do download é maior do que para o Blazor Server, depende do aplicativo e o processamento é totalmente feito no hardware do cliente. No entanto, esse tipo de aplicativo tem um tempo de resposta rápido. Como o próprio nome sugere, essa estrutura do lado do cliente é escrita em WebAssembly, ao contrário de JavaScript (embora possam ser usados juntos). Blazor WebAssembly 3.2.0 foi lançado em 19 de maio de 2020.
